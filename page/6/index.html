记录_基于UptimeRobot服务器状态监控的搭建-初始
发表于2020-10-14|工具|站点监控
在网上找了很多搭建方法一个基于 UptimeRobot API 的状态监控仪表盘(O’S tatus)： 原作者GitHub —— 站点演示但因为配置原因一直不成功，最终找到一个简单的。 申请uptime Robot的API 进入官网：https://uptimerobot.com 申请账号。 添加你需要监控的网站。 点击界面上面的My Settings，拉到最底部，API Settings这块点击Monitor-Specific API keys申请API的key。 下载网页配置文件及更改 下载github的文件点击下载 (下载1.3版本，新版本更新后没有配置成功)。 更改配置文件config.js。ApiKeys中的值为从uptime Robot中获取的key值。SiteUrl为你的网站连接。SiteName为网点名。其余按文件注释进行配置。 更换favicon.ico图标。 将文件部署到支持静态网页的服务器上。
